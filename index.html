Quick Sort Algorithm

Quicksort is a divide and conquer algorithm. Quicksort takes in an array of integers. Quicksort first divides a large array into two smaller sub-arrays: the low elements and the high elements. Quicksort can then recursively sort the sub-arrays. 

              
                def quick_sort(list)
                  raise 'Invalid input, expecting array of Ints'
                  if list.nil? || !list.is_a?(Array) || list.all? { |x| !x.is_a?(Integer) }
                  return list if list.empty?
                  pivot_index = (list.length / 2).ceil
                  pivot_value = list[pivot_index]
                  list.delete_at(pivot_index)

                  low = []
                  high = []

                  list.each do |x|
                    if x <= pivot_value
                      low << x
                    else
                      high << x
                    end
                  end
                  quick_sort(low) + [pivot_value] + quick_sort(high)
                end

Bubble Sort Algorithm

Bubble Sort is a compare sorting algorithm that repeatedly steps through the list to be sorted, compares each pair of adjacent items and swaps them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, which indicates that the list is sorted. Bubble Sort takes in an array of integers. 

              
              def bubble_sort(list)
                raise 'Invalid input, expecting array of Ints'
                if list.nil? || !list.is_a?(Array) || list.all? { |x| !x.is_a?(Integer) }
                n = list.length
                loop do
                  swapped = false
                  (n - 1).times do |i|
                    if list[i] > list[i + 1]
                      list[i], list[i + 1] = list[i + 1], list[i]
                      swapped = true
                    end
                  end
                  break unless swapped
                end
                list
              end

Factorial Challenge

This ruby program calculates the factorial of a number and only takes a number as an input. I demonstrate three different ways to approach this problem, with all three achieving the same result. 

              
                module Factorial
                  def self.of(number)
                    raise 'Invalid Input' if number.nil? || number.is_a?(String)
                    factorial = 1
                    count = number
                    while count > 0
                      factorial *= count
                      count -= 1
                    end
                    factorial
                  end

                  def self.of2(number)
                    raise 'Invalid Input' if number.nil? || number.is_a?(String)
                    (1..number).inject { |factorial, n| factorial * n }
                  end

                  def self.of3(number)
                    raise 'Invalid Input' if number.nil? || number.is_a?(String)
                    (1..number).reduce(1, :*)
                  end
                end

FizzBuzz Challenge

This ruby program prints the numbers from 1 to the input. For multiples of 3, it prints 'fizz' instead of the number, and for multiples of 5, it prints 'buzz'. For numbers which are multiples of both 3 and 5, it prints 'fizzbuzz'. 

              
                def fizzbuzz(number)
                  raise 'Invalid Input' if number.nil? || number.is_a?(String)
                  (1..number).each do |count|
                    if (count % 3).zero? && (count % 5).zero?
                      puts 'fizzbuzz'
                    elsif (count % 3).zero?
                      puts 'fizz'
                    elsif (count % 5).zero?
                      puts 'buzz'
                    else
                      puts count
                    end
                  end
                end
